Simple Sprinklr API Authorization Workflow

This app shows how to generate and process a user authentication process.

In the app, the API key and Secret must be set or retrieved from a secure location.
The first step is to generate the URL. It passes the API key and callback address on where to send the ticket (code parameter) that can be used to generate the access token.

The callback address should set to http://thisdomain.com/auth (or https: if you are using ssl)
The callback address must also be set correctly in the Application Settings at Sprinklr Developer Portal

Click here to begin the authorization process.
